# Memoria Clew

A local-first, memory-augmented research assistant.

## Overview

Memoria Clew is designed to help you capture, recall, and synthesize information for your projects.

## Getting Started

1.  Copy `.env.example` to `.env` and fill in the required values.
2.  Install dependencies:
    ```bash
    cd frontend && npm install
    cd ../server && npm install
    ```
3.  Run development servers:
    ```bash
    # Frontend
    cd frontend && npm run dev

    # Backend
    cd server && npm run dev
    ```

## Architecture

-   **Frontend**: React + Vite + Firebase (Auth, Firestore)
-   **Backend**: Node.js + Express + MCP (Model Context Protocol)
